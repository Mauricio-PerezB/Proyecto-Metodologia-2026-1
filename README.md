# Sistema de Gestión y Preinscripción

Bienvenido al repositorio del Sistema de Gestión. Esta es una plataforma web desarrollada para facilitar la administración y el control de usuarios, clases y evaluaciones. 

## Características principales
- **Formulario de Preinscripción:** Registro de nuevos postulantes con selección de plan y carga de comprobantes de pago.
- **Panel de Secretaría:** Gestión de solicitudes, validación de pagos, asignación de roles y control administrativo de vehículos.
- **Gestión de Clases y Exámenes:** Historial de clases teóricas/prácticas y control de calificaciones.
- **Envío automático de notificaciones:** Correos automatizados para la aceptación o rechazo de solicitudes.

## Tecnologías Utilizadas
- **Frontend:** React, Vite, Tailwind CSS / DaisyUI
- **Backend:** Node.js, Express, TypeORM
